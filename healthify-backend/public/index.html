Healthify · Backend UI

This page is your starting point. Use these links instead of typing any /api/... URLs.

User Area

Pages intended for normal users of the app.

  • User Home (categories, exercises, workouts, current workout)
  • User Signup
  • User Login

Admin Area

Pages intended for admins only. These use the admin JWT token for managing data.

  • Admin Home
  • Admin Login
  • Categories Admin
  • Exercises Admin
  • Workouts Admin

Health & Debug

Quick checks exposed via the UI; no need to call APIs manually.

  • Health status is shown on: admin-categories, admin-exercises, admin-workouts, and user-home pages.
  • All other functionality (CRUD for categories, exercises, workouts) is triggered from the above pages.
